First entry in the log. This site runs on Eleventy, builds to static HTML, and deploys to GitHub Pages on every push — so writing a post is just adding a markdown file to `src/posts/`.

The header globe reads straight from `src/_data/qsos.json`, so every contact logged there shows up as an arc automatically. Same data drives the [QSO log page](/log/) table.
